# JQuery spoiler

Spoiler is a jQuery plugin that hide and show some long text.

File structure:

  * `css/` - compiled CCS from `src-css`
  * `js/` - javascript plugin + demo script
  * `images/` - demo images
  * `demo.md` - demo code

Use:

    $('.js-spoiler').spoiler();

[Live demo with examples](http://js.pushorigin.ru/spoiler)
